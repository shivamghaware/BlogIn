# **App Name**: BlogIn

## Core Features:

- User Authentication: Secure sign-up, login, and profile management using Firebase Authentication.
- Post Creation: Rich text editor for creating and formatting blog posts.
- Post Management: Creating, reading, updating and deleting blog posts using Firestore.
- Tagging System: Adding and browsing posts by tags stored in Firestore.
- Real-time Updates: Display new comments immediately using Firestore.
- User interaction: Add comments, like posts and follow users.
- Suggest post category: AI Tool which suggests what category to tag the post.

## Style Guidelines:

- Primary color: A soft, desaturated blue (#64B5F6) to provide a calming and reliable feel.
- Background color: Off-white (#FAFAFA) for a clean, minimalist aesthetic.
- Accent color: Light green (#AED581) for interactive elements and highlights.
- Body and headline font: 'Literata', a serif font for readability and a literary feel.
- Simple, line-based icons for a clean and modern look.
- Clean, white space-heavy layout, similar to Medium.com.
- Subtle animations for transitions and user interactions to enhance user experience.
